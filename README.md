# EscrowLink
Freelancer escrow payments using Stellar

## Problem
Freelancers lose money due to lack of trust in payments

## Solution
Smart contract escrow releases funds only after approval

## Timeline
2–3 weeks MVP

## Features
USDC transfers, Soroban contracts

## Vision
Secure freelance economy in SEA

## Prerequisites
Rust, Soroban CLI

## Build
soroban contract build

## Test
cargo test

## Deploy
soroban contract deploy

## Example
soroban contract invoke --fn create_escrow
